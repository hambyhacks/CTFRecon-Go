# CTFRecon-Go

Go implementation of CTFRecon.sh

## To install

```bash
1. git clone https://github.com/hambyhacks/CTFRecon-Go
2. cd CTFRecon-Go/
3. go build .
```

CTFRecon-Go binary should be created after using ``go build .``
